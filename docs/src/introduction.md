# LIAO Documentation Overview

This repository contains the official documentation for the LIAO (Liga Acadêmica de Inteligência Artificial e Otimização) full-stack platform.

## Target Audience & Usage

The documentation serves two distinct interfaces:

### 1. Human Contributors
Provides architectural technical specifications, design system constraints, API type synchronization rules, and local development setup instructions.

### 2. LLMs and Automated AI Coding Agents
Serves as an authoritative context source to enforce structural and stylistic constraints. AI tools (e.g., Cursor, Gemini, Claude) must inspect these rules prior to generating code to prevent style regressions or non-standard token usage.
